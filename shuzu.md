# 数组
> 存储一组相关数据的容器
* 数组声明/创建
    * var arr=[]
    * var arr=new Array（）
    >结果是完全相同的
* 数组的赋值
    * 在声明的同时赋值
        * var arr=[1,2,3,4,5];
        * var arr=new Array(1,2,3,4,5);
        > new Array() 只有一个参数，这个数字表示的就是数组的长度
    * 通过下标赋值
        * arr[0]=1；arr[1]=2；
* 数组的length
    * 数组中下标最大的数的下标加一
* 数组的下标
    * 数组中每一个数对应的位置 从0开始
* 数组的访问
    * 通过数组的下标
    > console.log(arr[0]);
* 数组的遍历/枚举
    * for(){}  while(){}
* 二维数组
    * 数组中所有的元素
